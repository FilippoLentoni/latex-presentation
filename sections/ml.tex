% ---------------- Slide 7 ----------------
\begin{frame}{2) ML Systems: What Changes?}
\begin{itemize}
  \item Behavior is learned from data (not explicitly coded)
  \item Correctness is statistical: performance depends on the data distribution
  \item Generalization matters: in-sample vs out-of-sample
  \item Train/validation/test splits; cross-validation
  \item Metrics: MSE, RMSE, accuracy, precision/recall, ..
  \item Failure modes: overfitting, leakage, distribution shift
\end{itemize}
\end{frame}
